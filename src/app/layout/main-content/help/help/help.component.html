ChEMBL-explorer help & FAQ

1. Side panel (search and settings)

At the side panel, user can perform target/molecule keyword search, molecule structure/substructure search, browse targets by protein class or KEGG disease class. And at the setting tab, there are global options for the tool, however, only tooltip toggle available for the current version, more options will available in future.

2. Target page

ChEMBL-explorer extended official target page with more chart widgets, MMP, target network and scaffold, etc. Click corresponding tab to view them. It may take seconds to load for targets with abundant bioactivity data. (This is why we recommend user to build their local version.)

FAQ

1. What is target network?
A edge in target network denotes that the two targets linked by it shared above a certain number of active compounds, here the activity threshold and the shared number are tunable. By default ChEMBL-explorer will set default thresholds according to the size of network. 2. What is scaffold activity?
In the first, scaffold here means BM scaffold (Bemis, G. W., & Murcko, M. A. (1996), dio: 10.1021/jm9602928). And the scaffold activity of a target is aggregated (mean, maximum, minimum and median) from the activities of all molecules that active against the target and belong to the scaffold. 3. What is target scaffold network?
A edge in target network denotes that the two targets linked by it shared above a certain number of active scaffolds, here the activity threshold and the shared number are tunable. By default ChEMBL-explorer will set default thresholds according to the size of network. 4. How to view corresponding data of a chart?
On the assay type, activity type, publication and molecule properties histogram charts, double click your interest part (eg. the number rotate bonds bar of 4, or all 56 publications of Tyrosine-protein kinase ABL in 2013) will navigate to corresponding data table. 5. What are related molecules?
Related molecules are a set molecules that shared more than 10 active targets with current molecule. 6. Have display problems?
ChEMBL-explorer was only test on the latest version of browser, so please try the latest browser (google chrome , firefox, etc) if you have any display problems. Have a question?
