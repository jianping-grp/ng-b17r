ChEMBL-explorer help & FAQ

1. Side panel (search and settings)

At the side panel, user can perform target/molecule keyword search, molecule structure/substructure search, browse targets by protein class or KEGG disease class. And at the setting tab, there are global options for the tool, however, only tooltip toggle available for the current version, more options will available in future.

2. Target page

ChEMBL-explorer extended official target page with more chart widgets, MMP, target network and scaffold, etc. Click corresponding tab to view them. It may take seconds to load for targets with abundant bioactivity data. (This is why we recommend user to build their local version.)
