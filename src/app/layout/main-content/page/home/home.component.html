ChEMBL-explorer

ChEMBL-explorer architecture

ChEMBL-explorer is a web application for interactive visualization and exploration of large-scale bioactivity data in ChEMBL database . ChEMBL-explorer was developed not intend to be a substitution of the official ChEMBL web site, but a extensive use of ChEMBL by discovering patterns in large-scale bioactivity data, and provides intuitive visualization of bioactivity data. We have open sourced ChEMBL-explorer, so that user who interest in building a local version or getting involved in developing can found both the frontend and the backend at our github repository(ng-chembl and chembl_explorer).

Your feedback is very helpful, please share your opinion and thoughts with us. email zhonghua.wang(AT)mail.nankai.edu.cn bug_report Report a bug about the backend or the frontend.
