Bioactivity-explorer

Bioactivity-explorer architecture

Bioactivity-explorer is a web application for interactive visualization and exploration of large-scale bioactivity data in ChEMBL database (V24) . Bioactivity-explorer was developed not intend to be a substitution of the official ChEMBL web site, but a extensive use of ChEMBL by discovering patterns in large-scale bioactivity data, and provides intuitive visualization of bioactivity data. We have open sourced Bioactivity-explorer, so that user who interest in building a local version or getting involved in developing can found both the frontend and the backend at our github repository(ng-b17r and b17r).
